# 客户服务进度看板 - 开发计划

## 文件清单 (MVP)

1. **src/pages/ServiceBoard.tsx** - 主看板页面，包含：
   - 列表视图（默认）
   - 看板视图（Kanban按服务阶段分栏）
   - 员工视图（按运营负责人分组）
   - 快捷标签筛选（全部、今日待处理、已逾期、有问题卡点、等待客户资料、即将到期、本周有更新、长期未更新）
   - 综合筛选（客户名称、行业、国家、州、城市、负责人、服务类型、服务阶段、问题状态等）
   - 服务进度详情弹窗（含任务清单、卡点信息、时间线）
   - 新增/编辑服务进度
   - 新增/编辑服务任务
   - 老板统计概览面板
   - 运营工作量统计

2. **src/lib/service-board-config.ts** - 服务看板配置常量
   - 服务阶段定义（通用、新媒体、网站/点餐）
   - 阶段对应进度百分比
   - 任务类型定义
   - 问题状态定义
   - 标签映射

3. **更新 src/App.tsx** - 添加 /service-board 路由
4. **更新 src/components/Layout.tsx** - 添加导航菜单项
5. **更新 src/lib/permissions.ts** - 添加服务看板权限配置

## 权限设计
- 管理员/老板：查看全部，可编辑
- 运营：查看自己负责的客户，可更新任务和进度
- 销售：查看自己客户的服务进度，不可修改运营内容
- 财务：查看服务状态和到期状态，不可修改
